# Hello_Quantum_World

> Just a simple Hello World program written in Qsharp and Python.

## Prerequisites  
- The code requires [Quantum Development Kit](https://docs.microsoft.com/en-us/quantum/?view=qsharp-preview) SDK and need to be installed.  
- To execute the python code, we need qsharp module. To install it use the command ```pip install qsharp```.
